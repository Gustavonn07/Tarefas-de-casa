Object.keys(), Object.values() e
Object.entries()

O que são esses métodos?

  • Os métodos em JavaScript são funções que estão associadas a um objeto e podem ser chamadas para realizar ações específicas ou executar tarefas relacionadas ao objeto em questão. Eles desempenham um papel fundamental na programação orientada a objetos, permitindo a organização lógica e modularização do código.
  • Os métodos permitem organização e a manutenção do código mais facilmente
  • Object.entries, Object.keys e Object.values são métodos estáticos do objeto Object no JavaScript. Eles são usados para trabalhar com objetos.

O que esses métodos fazem?

  • Object.keys

    Retorna uma array com as keys dos objetos.

  • Object.values

    Retorna uma array com os valores das keys dos objetos.

  • Object.entries

    Retorna uma array com as keys e valores dos objetos.
